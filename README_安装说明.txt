  快手图标屏保 - 安装说明
═══════════════════════════════════════════════════════════

⚠️ 重要：如果提示"无法打开，因为 Apple 无法检查其是否包含恶意软件"

这是 macOS 的正常安全提示，请按以下方法解决：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法一：右键打开（最简单，推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 不要直接双击 .saver 文件
2. 右键点击 KuaiShouIconScreenSaver.saver
3. 选择"打开"
4. 在弹出的提示中点击"打开"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法二：系统设置中允许
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 双击 .saver 文件后，如果提示无法打开
2. 打开"系统设置" → "隐私与安全性"
3. 在底部找到提示信息，点击"仍要打开"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法三：终端命令（适合熟悉命令行的用户）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

打开"终端"，运行：

sudo xattr -rd com.apple.quarantine ~/Downloads/KuaiShouIconScreenSaver.saver

（将路径替换为你实际的文件路径）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
安装步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 使用上述任一方法解决安全提示
2. 双击文件，系统会自动安装
3. 打开"系统设置" → "桌面与程序坞" → "屏幕保护程序"
4. 在列表中选择"KuaiShouIconScreenSaver"
5. 点击"屏幕保护程序选项"可以配置动画开关

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 重要：如果安装后无法预览或使用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果安装后仍然提示"无法打开，因为 Apple 无法检查其是否包含恶意软件"，
说明需要移除隔离属性。请在终端运行以下命令：

sudo xattr -rd com.apple.quarantine ~/Library/Screen\ Savers/KuaiShouIconScreenSaver.saver

然后重新打开系统设置，就可以正常预览和使用了。

常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 为什么会有这个安全提示？
A: 因为文件没有经过 Apple 官方公证，这是正常的。只需要允许即可。

Q: 这个文件安全吗？
A: 是的，这是开源项目，代码完全透明。

Q: 安装后找不到屏保？
A: 确保文件已正确安装。如果还是没有，尝试重启电脑。

Q: 安装后无法预览或使用，仍然提示安全警告？
A: 这是因为隔离属性没有被移除。请在终端运行：
   sudo xattr -rd com.apple.quarantine ~/Library/Screen\ Savers/KuaiShouIconScreenSaver.saver
   然后重新打开系统设置即可。

Q: 有没有更简单的安装方法？
A: 可以使用桌面上的"安装屏保.sh"脚本，将 .saver 文件拖拽到脚本上即可自动安装。
