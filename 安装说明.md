# 快手图标屏保安装说明

## 如果遇到"无法打开，因为 Apple 无法检查其是否包含恶意软件"

这是 macOS 的安全保护机制（Gatekeeper），请按照以下任一方法解决：

### 方法一：右键打开（推荐，最简单）

1. **不要直接双击** `.saver` 文件
2. **右键点击** `KuaiShouIconScreenSaver.saver` 文件
3. 选择 **"打开"**
4. 在弹出的安全提示中，点击 **"打开"**

这样就能绕过 Gatekeeper 的限制。

### 方法二：系统设置中允许

1. 双击 `.saver` 文件后，如果提示"无法打开"
2. 打开 **"系统设置"**（或"系统偏好设置"）
3. 进入 **"隐私与安全性"**（或"安全性与隐私"）
4. 在底部找到提示信息，点击 **"仍要打开"** 按钮

### 方法三：使用终端命令（适合熟悉命令行的用户）

打开"终端"应用，运行以下命令：

```bash
# 移除隔离属性
sudo xattr -rd com.apple.quarantine ~/Downloads/KuaiShouIconScreenSaver.saver

# 然后就可以正常双击打开了
```

注意：将路径替换为你实际的文件路径。

### 方法四：临时禁用 Gatekeeper（不推荐，仅用于测试）

```bash
sudo spctl --master-disable
```

⚠️ **警告**：这会降低系统安全性，使用后记得重新启用：
```bash
sudo spctl --master-enable
```

## 安装步骤

### 方法一：使用安装脚本（推荐）

1. 下载 `KuaiShouIconScreenSaver.saver` 和 `安装屏保.sh` 文件
2. 将 `.saver` 文件拖拽到 `安装屏保.sh` 脚本上
3. 或在终端运行：`bash 安装屏保.sh ~/Downloads/KuaiShouIconScreenSaver.saver`
4. 脚本会自动移除隔离属性并安装
5. 打开 **"系统设置"** → **"桌面与程序坞"** → **"屏幕保护程序"**
6. 在列表中选择 **"KuaiShouIconScreenSaver"**

### 方法二：手动安装

1. 下载 `KuaiShouIconScreenSaver.saver` 文件
2. 使用上述任一方法解决安全提示
3. 双击文件，系统会自动安装到 `~/Library/Screen Savers/`
4. **重要**：如果安装后无法预览或使用，请在终端运行：
   ```bash
   sudo xattr -rd com.apple.quarantine ~/Library/Screen\ Savers/KuaiShouIconScreenSaver.saver
   ```
5. 打开 **"系统设置"** → **"桌面与程序坞"** → **"屏幕保护程序"**
6. 在列表中选择 **"KuaiShouIconScreenSaver"**
7. 点击 **"屏幕保护程序选项"** 可以配置动画开关

## 常见问题

**Q: 为什么会有这个安全提示？**  
A: 因为屏保文件没有经过 Apple 的官方公证（Notarization）。这是正常的，只需要按照上述方法允许即可。

**Q: 这个文件安全吗？**  
A: 是的，这是开源项目，代码完全透明。如果你担心，可以查看源代码。

**Q: 安装后找不到屏保？**  
A: 确保文件已正确安装到 `~/Library/Screen Savers/` 目录。如果还是没有，尝试重启电脑。

**Q: 可以删除这个文件吗？**  
A: 可以。在"系统设置"中移除屏保，然后删除 `~/Library/Screen Savers/KuaiShouIconScreenSaver.saver` 文件夹即可。
